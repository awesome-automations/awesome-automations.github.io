Jinja Code

Jinja2 is a templating language for python, and is used heavily in the Home Assistant. There are a few ways you can run jinja code.

  • Using python directly
  • Using Home Assistant's Template Editor

If you are testing plain jinja logic, you can test it using either options. But, if you are testing jinja code that uses Home Assistant's domain objects, you have to test it using the template editor.

You can find that template editor by clicking on the hamburger icon on the top left of your Home Assistant instance, Under Developer Tools, click on Templates icon. You will see a page with some code.

You can safely delete that code and paste any jinja2 code there, it should render the output on the right side of the page.
